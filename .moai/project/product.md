# PandaTV KPI Dashboard — Desktop Web Prototype

## 프로젝트 개요

**목적:** 더블미디어 사업기획팀장 면접용 비기능 프로토타입
**버전:** Web (Desktop) — Mobile PRD의 확장 버전
**출력:** 단일 React 파일 (`.jsx`) — 데스크톱 우선, 다크 모드, 정적 샘플 데이터 전용
**타겟 뷰포트:** 1440px (최소 1024px 반응형)

## 미션

라이브 스트리밍 플랫폼의 KPI 사고 구조를 체계적으로 보여주는 데스크톱 대시보드 프로토타입을 구축한다. 면접 현장에서 노트북으로 시연하여 사업기획 리더의 데이터 분석 역량과 전략적 사고를 입증한다.

## 비전

사업기획팀장 후보로서 팬더티비(PandaTV) 플랫폼의 핵심 KPI를 구조화하고, 데이터 기반 전략적 기회를 식별·제안할 수 있는 역량을 시각적으로 증명하는 인터뷰 도구.

## 사용자 페르소나

| 페르소나 | 설명 |
|---------|------|
| 면접관 (더블미디어 경영진) | 대시보드를 통해 후보자의 KPI 구조화 능력과 사업 이해도를 평가 |
| 프레젠터 (면접 후보자 본인) | 노트북에서 직접 시연하며 사업기획 역량을 어필 |

## 핵심 기능

### Dashboard 탭
- 8개 KPI 요약 카드 (MAU, DAU, 신규가입, 30D 리텐션, 후원매출, 결제유저, ARPPU, 활성 BJ)
- 플랫폼 성장 추이 차트 (MAU + 후원매출 복합 차트)
- 비즈니스 헬스 요약 (4개 차원: 유저성장/수익화/BJ공급/리스크운영)
- 유저 전환 퍼널 (방문 → 가입 → 시청 → 첫결제 → 재구매)
- 리텐션 헬스 패널
- BJ 공급 건강도 (매출 집중도 도넛 + 티어별 분포)
- 매출 구조 (후원 추이, 결제유저 추이, 매출 믹스)
- 리스크 & 운영 거버넌스
- 핵심 인사이트 3개 + 추천 액션 3개

### Opportunities 탭
- 3개 전략 이슈 카드 (BJ 집중 리스크, 첫결제 전환율, 운영 스케일링)
- 2개 성장 베팅 (크리에이터 수익화 인프라, 크리에이터 스튜디오/라운지)

## 비즈니스 목표 & 성공 지표

| 지표 | 목표 |
|------|------|
| 면접 시연 | 1440px 뷰포트에서 스크롤 없이 헤더 + 8개 KPI + 트렌드 차트 상단 노출 |
| 코드 품질 | 단일 파일, 섹션 주석, 콘솔 에러 0 |
| 데이터 정합성 | 모든 수치가 공식 앵커에서 파생·검증 가능 |

## 경쟁 분석 (프로토타입 차별화)

- 단순 목업이 아닌 **인터랙티브 차트** (Recharts 툴팁)
- 모바일 + 데스크톱 양방향 프로토타입으로 멀티 플랫폼 사고 입증
- "Planning hypothesis" 섹션으로 전략적 사고 역량 시각화

## Mobile PRD와의 관계

| 차원 | Mobile (Snapshot) | Web (Dashboard) |
|------|-------------------|-----------------|
| 용도 | 이동 중 면접 참고 | 노트북 데스크 프레젠테이션 |
| KPI 카드 | 6개 | 8개 (DAU, ARPPU 추가) |
| 차트 | 2개 | 6개 (line, bar, funnel, pie, retention) |
| 레이아웃 | 단일 컬럼 스크롤 | 멀티 컬럼 그리드 |
